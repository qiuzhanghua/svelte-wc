# Svelte + TS + Vite + Web Components
Use svelte to develop web components, and vite to build and serve.

## installation
```bash
pnpm install
```

## build
```bash
pnpm build-all

```

## development
```bash
pnpm dev
```
and visit http://localhost:5173
